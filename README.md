Proyecto sobre BlackJack para la asignatura de [SIS-112] PROGRAMACIÓN I 
Javier Beltrán Aparicio
El proyecto cuenta con todas las funciones de el juego de BlackJack, e incluye una baraja completa de 52 cartas para jugar al juego,
propone como objetivo alcanzar una meta de 3500 monedas, permitiendo tambien al usuario escoger la cantidad de monedas con las que
inicia de modo que haga el juego más largo, el juego termina cuando el usuario llega a la meta propuesta o cuando pierde todas sus 
monedas, también durante cualquier momento de la partida el usuario puede reestablecer sus monedas, reiniciando así el juego.
